---

## 📖 Overview

Smart Campus is a web‑based university management platform built with Spring Boot that digitalizes campus operations including student management, facility booking, course management, and campus announcements.

## 🛠 Technologies Used

- **Spring Boot** – backend framework
- **Java** – programming language
- **MySQL** – relational database
- **Thymeleaf** – server‑side template engine
- **HTML5 / CSS3** – frontend markup and styling
- **JavaScript** – client‑side interactivity
- **Maven** – dependency management and build tool

## ✨ Key Features

- 👨‍🎓 Student and faculty management
- 📚 Course and timetable management
- 🏛 Campus facility booking system
- 📢 Announcements and notice board
- 📊 Admin dashboard with analytics
- 📱 Responsive web interface (mobile‑friendly)
